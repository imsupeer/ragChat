Retrieval uses dense similarity search in ChromaDB as the default baseline. When ENABLE_HYBRID is true, the system also runs local BM25 keyword search and merges results with Reciprocal Rank Fusion. The system can restrict search to selected document_ids, then retrieves top_k chunks and truncates again to max_context_chunks before prompt construction. Optional heuristic reranking can reorder candidates when ENABLE_RERANKING is true.
